# Home-Lab-Network-Infrastructure-Proxmox-pfSense-VLAN-Segmentation-Network-Security
Built a self-hosted enterprise-style home lab using Proxmox, pfSense, and a Cisco Catalyst 2960 to develop practical network engineering skills. Implemented VLAN segmentation, firewall policy enforcement, management isolation, SSH hardening, and resolved real-world routing, DHCP, and network security issues.
